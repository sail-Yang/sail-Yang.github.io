1、将下面程序编译、链接，然后用Debug加载、跟踪，回答问题

代码

assume cs:code,ds:data,ss:stack
data segment
        dw 0123h,0456h,0789h,0abch,0defh,0fedh,0cbah,0987h
data ends
stack segment
        dw 0,0,0,0,0,0,0,0
stack ends

code segment
start: mov ax,stack
       mov ss,ax
       mov sp,16

       mov ax,data
       mov ds,ax

       push ds:[0]
       push ds:[2]
       pop ds:[2]
       pop ds:[0]

       mov ax,4c00h
       int 21h
code ends

end start

问题

  1. 程序返回前，data段的数据为多少？
  2. CPU执行程序，程序返回前，cs = ? ,ss = ?,ds = ?
  3. 设程序加载后,code段的段地址为X，则data段的段地址为?stack段的段地址为?

解答

  1. data段的数据一直没变，开始是多少，结束时就是多少：

  2. cs = 076C,ss = 076B,ds = 076A

  3. 从第二问我们就可以看出来，如果code段的段地址为X ，则data段的段地址为X - 2,stack段的段地址为X - 1

2、将下面程序编译、链接，然后用Debug加载、跟踪，回答问题

代码

assume cs:code,ds:data,ss:stack
data segment
        dw 0123h,0456h
data ends
stack segment
        dw 0,0
stack ends

code segment
start: mov ax,stack
       mov ss,ax
       mov sp,16

       mov ax,data
       mov ds,ax

       push ds:[0]
       push ds:[2]
       pop ds:[2]
       pop ds:[0]

       mov ax,4c00h
       int 21h
code ends

end start

问题

  1. 程序返回前，data段的数据为多少？

  2. CPU执行程序，程序返回前，cs = ? ,ss = ?,ds = ?

  3. 设程序加载后,code段的段地址为X，则data段的段地址为?stack段的段地址为?

  4. 对于如下定义的段，如果段中的数据占N个字节，则程序加载后，该段实际占有的空间为？

    name segment
……
name ends

解答

  值得注意的地方是，我们这次定义的数据端和栈段虽然都是4字节的，但是我们实际观察就会发现，编译器还是会给我们每个段分配16个字节的空间：

  076A：00~076A：0F是数据段，可以看到编译器给data段分配了16个字节的空间，不足补0；076A：10~076A：1F（即076B：00~076B：0F）是栈段，可以看到编译器给栈段也分配了16个字节的空间。

  1. data段的数据一直没变，开始是多少，结束时就是多少。

  2. cs = 076C,ss = 076B,ds = 076A

  3. 从第二问我们就可以看出来，如果code段的段地址为X ，则data段的段地址为X - 2,stack段的段地址为X - 1

  4. 若N 能被16整除，则该段占有空间为(N/16)*16字节；若N不能被16整除，则该段占有空间为(N/16+1)*16字节。

        dw 0,0

问题

  1. 程序返回前，data段的数据为多少？
  2. CPU执行程序，程序返回前，cs = ? ,ss = ?,ds = ?
  3. 设程序加载后,code段的段地址为X，则data段的段地址为?stack段的段地址为?

解答

  不想写了，和上面一个意思，就是把段地址换一下。

4、如果将end start这条命令改成end 那么上面三大题1、2、3哪一个还可以执行？

  我们之前说过end除了可以结束程序外，还可以指明程序的入口。

  现在将end start换成end，实际上就是不指明程序的入口，那么程序将直接顺序执行。这样的后果是，即使程序的第一个段不是代码段，程序还是会从第一个段开始执行，这就会导致出错。

  所以上面三大题中只有第三个还可以执行，因为第三个程序的第一个段就是代码段，所以不会出错。

5、编写code段中代码，将a段和b段中的数据依次相加，将结果存到C段中

附注：下面只写code segment 的内容

解答（循环两次，不是最优解）

code segment
start:
        mov ax,a
        mov ds,ax

        mov ax,b
        mov es,ax

        mov cx,8
        mov bx,0

    s:  mov ax,[bx]
        add es:[bx],ax
        inc bx
        loop s

        mov ax,c
        mov ds,ax

        mov bx,0
        mov cx,8

    s1:  mov ax,es:[bx]
        add [bx],ax
        inc bx
        loop s1

        mov ax,4c00h
        int 21h
code ends
end start

6、程序如下，编写code段中代码，用PUSH将A段中的前8个字型数据，逆序存储到B段

assume cs:code
a segment
    dw 1,2,3,4,5,6,7,8
a ends

b segment
    dw 0,0,0,0,0,0,0,0
b ends

code segment
    ???
code ends

end start

解答

code segment
    mov ax,a
    mov ds,ax
    mov ax,b
    mov ss,ax

     mov bx,0
     mov cx,8
