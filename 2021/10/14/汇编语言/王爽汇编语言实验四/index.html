王爽《汇编语言》实验四

发表于2021-10-14|更新于2022-01-16|汇编语言8086实验
|字数总计:152|阅读量:

题目

  这第二问就是第一问的优化版，所以我们直接写第二问了，第一问就不写了，第三问没啥好讲的的，跟着调试就行了，所以第三问也不写了。

解答

assume cs:code
code segment
    mov ax,0020h
    mov ds,ax
    mov cx,64
    mov bx,0

  s:mov ds:[bx],bx
      inc bx
      loop s

      mov ax,4c00h
      int 21h
code ends
end

​ 值得注意的一个问题是，0:200~0:23F等同于0200:0~0200:3f,推荐使用后面这种写法，后面这种写法使得我们用一个bx就能搞定偏移地址和数字的递增。
