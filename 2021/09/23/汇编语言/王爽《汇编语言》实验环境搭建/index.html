|字数总计:873|阅读量:

前言

  这一学期学校的课程里有汇编语言，要求是掌握8086的，我上网搜了一下，网评比较好的教材是王爽的《汇编语言（第四版）》，于是我就选择了这本书作为我的主力教材，而学校的教材准备考试前用一用。

教材的电子版推荐去Z-Library下载，这个真的好用，我的书全是这个上面下的。

  这个专题主要是存放王爽《汇编语言》这本书里的一些实验记录以及汇编语言的使用问题等等，至于笔记什么的，我都放到我的语雀里了，就不展示了，网上有很多关于这本书的笔记。

实验环境的搭建

  首先我们要知道要实践8086汇编语言的学习，最好的学习环境是DOS操作系统，Windows10是不支持DOS的，我选择的方案是Dosbox + MASM,当然也可以用VMware搭个虚拟机，我这里就稍微介绍一下Dosbox+MASM搭建实验环境的方法。

  首先，我们要知道Dosbox和MASM是什么，咱们长话短说，简单说一说。

Dosbox是一个模拟器，模拟的是老古董：ms-Dos系统，它能为本地的Dos程序提供执行环境，其早期主要是为了玩Dos游戏设计的，用C++编写。MASM 是 Microsoft Macro Assembler 的缩写，它是微软为 x86 微处理器家族所写的一套宏汇编器。它最初是用来发展在 MS-DOS上面运行的软件。
