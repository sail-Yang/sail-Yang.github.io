指针基础知识

机器级上指针表示的内容

现代计算机大多把内存分割为字节，而每一字节存储8位信息。每个字节都有唯一的地址，就像我们的学号一样，用来和其它内存区别。

可执行程序由代码和变量组成。程序中的每个变量占有一个或多个字节内存，把第一个字节的地址称为变量的地址。就像下图，变量i占有3001,3002两个字节，所以变量i的地址为3001.

我们虽然用数表示内存地址，但是地址的取值范围是不同于整数的范围的，所以一定不能用普通整型变量来存储地址。但是，可以用特殊的指针变量来存储地址。这就是指针的出处。**指针就是地址，而指针变量就是存储地址的变量。**我们可以用下面这种方式来表示，指针变量p存储变量i的地址：

指针变量的声明

常见的指针声明形式：

  1. int p 这是一个普通的整型变量；int *p 首先从p开始，先与*结合，说明p是一个指针，然后再与int结合，说明指针的类型为Int型，所以p（指针变量名，不是*p）是一个返回整型数据的指针。
  2. int p[3]说明p是一个由整型数据组成的数组；int *p[3] p先与[3]结合，因为[]优先级>*，说明p是一个数组，然后再与*结合，说明数组里的元素都是指针，然后再与int 结合，说明指针指向的是对象是整型的，所以p是一个由返回整型数据的指针组成的数组。
  3. int (*p)[3] p先与*结合，说明p是一个指针，再与[3]结合，说明指针指向一个数组，再与Int结合，说明p指向一个由整型数据组成的数组。
  4. int **p 指向指针的指针。
  5. int p(int) p是一个函数，返回整型数据；int (*p)(int)，p是一个指向有一个整型参数且返回类型为整型的函数的指针。

指针的四方面

指针有四个方面的内容需要搞清楚：

指针的类型

从语法角度，我们只要把指针名字去掉，剩下的就是指针的类型（比如上面都把p去掉）。

指针所指向的类型

当通过指针来访问指针所指向的内存区时，指针所指向的类型决定了编译器将把那片内存区里的内容当做什么来看待。从语法看，去除名字和名字左边的*就行。

一定要注意指针所指向的类型和指针的类型的区别。

指针的值—地址

指针的值时指针变量存储的数值，这个数值将被编译器当作一个地址，而不是一个一般的数值。

前面也说了，变量i的地址用首地址来表示，我们说一个指针的值是XX，就相当于说该指针指向了以XX为首地址的一片内存区域。

指针本身所占据的内存区

用sizeof就可以测一下指针本身所占的内存了。

每当遇到一个指针，就问问自己这个指针的这四方面分别是什么，想的多了也就分的清了。

取地址运算符和间接寻址运算符

  • 为了找到变量的地址，可以使用取地址运算符&。举个例子，若x是一个变量，&x就是x在内存中的地址。

  • 为了获得指针所指向对象的访问，可以使用*（间接寻址运算符）。如果p是指针，*p表示p当前所指向的对象。

    &与*是一对逆运算。
