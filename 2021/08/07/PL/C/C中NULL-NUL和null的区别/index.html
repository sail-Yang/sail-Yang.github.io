CдёӯNULL,NULе’Ңnullзҡ„еҢәеҲ«

еҸ‘иЎЁдәҺ2021-08-07|жӣҙж–°дәҺ2022-02-14|PLC
|еӯ—ж•°жҖ»и®Ў:308|йҳ…иҜ»йҮҸ:

вҖғвҖғд№ӢеүҚеңЁзңӢеҲ«дәәд»Јз Ғж—¶пјҢзңӢеҲ°еҲ«дәәз”ЁдәҶдёӘNULпјҢжҲ‘еҝғйҮҢзәій—·пјҡCиҜӯиЁҖдёӯз©әжҢҮй’ҲдёҚжҳҜз”ЁNULLиЎЁзӨәеҗ—пјҹNULжҳҜд»Җд№Ҳй¬јиҘҝдёңпјҹжҲ‘жҠҠд»–зҡ„д»Јз ҒиҝҗиЎҢдёҖдёӢпјҢеҸ‘зҺ°ж №жң¬иҝҗиЎҢдёҚдәҶвҖҰвҖҰеҝғйҮҢеңЁжғіиҝҷе®¶дјҷжҳҜдёҚжҳҜе°‘жү“дәҶдёҖдёӘLрҹӨЈпјҢзңӢеҲ°д»–еҗҺйқўйғҪиҝҷд№ҲеҶҷпјҢжҲ‘еҝғйҮҢе°ұдә§з”ҹдәҶз–‘жғ‘вҖҰвҖҰ

вҖғвҖғд»ҠеӨ©еңЁзңӢд№Ұзҡ„ж—¶еҖҷпјҢзңӢеҲ°дәҶиҝҷд№ҲдёҖж®өиҜқпјҡ

вҖғвҖғжҲ‘ж“ҰпјҢиҝҳзңҹжңүNULиҝҷз§ҚеҶҷжі•вҖҰвҖҰжҲ‘еҸҲеҺ»жҹҘдәҶдёҖдёӢпјҢеҸ‘зҺ°NULе…¶е®һе°ұжҳҜвҖҷ\0вҖҷзҡ„ж„ҸжҖқрҹҳ…пјҢе…¶жң¬иҙЁжҳҜеӯ—з¬ҰпјҲcharacterпјүгҖӮ

вҖғвҖғиҖҢжҲ‘д»¬зҡ„иҖҒзҶҹдәәNULLе‘ўпјҢе…¶е®һжҳҜе®Ҹе®ҡд№үпјҢеұ•ејҖд№ӢеҗҺе‘ўжҳҜз©әжҢҮй’ҲеёёйҮҸгҖӮеңЁCиҜӯиЁҖдёӯпјҢNULLеёёеёёиў«дёӢйқўиҝҷж ·е®ҡд№үпјҡ

#define NULL    ((void *)0)

вҖғвҖғд№ҹе°ұжҳҜиҜҙNULL жҳҜдёҖдёӘ void *зұ»еһӢзҡ„жҢҮй’Ҳ(pointers)гҖӮ

вҖғвҖғжіЁж„ҸпјҢNULLе’ҢnullеҸҲжҳҜдёҚдёҖж ·зҡ„пјҢnullжҳҜдёҖз§ҚжҰӮеҝөзҡ„з»ҹз§°пјҢдёҚжҳҜCж ҮеҮҶйҮҢзҡ„жҢҮй’ҲеёёйҮҸжҲ–иҖ…еӯ—з¬ҰгҖӮ

вҖғвҖғжңҖеҗҺпјҢеҶҚжқҘиҜҙдёҖдёӘйӘҡж°”зҡ„йҡҗе–»рҹҳҸпјҡ

еҪ“жңүдәәз»ҷдҪ еҸ‘nullжҳҜд»Җд№Ҳж„ҸжҖқе‘ўпјҹ

вҖғвҖғжҲ‘д»¬зҹҘйҒ“пјҢдёҮзү©зҡҶеҸҜеҜ№иұЎпјҢnullеңЁжңүдәӣиҜӯиЁҖйҮҢе°ұиЎЁзӨәеҜ№иұЎиҝҳжңӘиөӢеҖјпјҢеҜ№иұЎдёәз©әпјҢеҸҜд»ҘзҗҶи§Јдёә"жІЎжңүеҜ№иұЎ"гҖӮ
