解题思路：（C）

- 建立一个字符串数组zimu{"ling","yi","er","san","si","wu","liu","qi","ba","jiu"}，下标与拼音一一对应。
- 计算n的各位数字之和sum。
- 将sum各位存入数组。假设sum = 123,则存入数组为[3,2,1]
- 将数组从后往前读，读到i,则输出zimu[i]

注意点：

- 可以看到输出样例很长很长，这个时候可能会想到用long long int，但是很明显是不行的，看看n的最大值可以是10^100(我一开始没注意)，所以**用字符串来读入n比较合适**。
- 我们现在得到用字符串num[]表示的n，注意字符串一个单元存储（num[i]）的**是字符而不是字符串**。
  - 将<u>字符</u>转化为int用num[i] - '0',比如我现在想取n左边第一位的Int型，我们就可以用num[0] - '0'
  - 将<u>字符串</u>转化为int可以用<stdlib.h>头文件里的atoi()函数
- 后面一个切sum的时候可以采取和前面同样切的方法，我这里其实有点多此一举了，也就是不用切字符串的方法。
- **注意输出的空格问题**。

代码：

```c
#include <stdio.h>
#include <string.h>
#include <stdlib.h> 
#define MAX_SIZE 1000
int main(){
    char *zimu[] = {"ling","yi","er","san","si","wu","liu","qi","ba","jiu"};
    char num[MAX_SIZE];
    int sum=0,i,count,k,sum_arr[20];
    
    gets(num);
    for(i=0;i<strlen(num);i++){
    	k = num[i]-'0';
    	sum += k;
	}
	while(sum>0)
    {
        k = sum%10;
        sum_arr[count]=k;
        sum = sum / 10;
        count += 1;
    }
    for(k = count-1;k>=0;k--){
        printf("%s",zimu[sum_arr[k]]);
        if (k != 0)
            printf(" ");
    }
    return 0;
}
```
