##  前言 

我们在前面提到过十大原理中的：**贸易可以使每个人的状况都变得更好**。为什么如此呢？

<!--more-->

在本书中会不断提到这个问题，在这一篇中我们将为之后的探讨打下一个基础。人们在相互交易时获得了什么好处？为什么人们要相互依存？

我们先假设一个简单的模型，假设这世上就两种食物：土豆和猪肉；而且只有两人：养猪张三和种土豆的李四。那么很明显，下面两种情况使得张三和李四不得不相互依存：

- 张三只会养猪，李四只会种土豆。如果它们不想只吃自己的猪肉或者土豆的话，他们就得贸易。
- 张三、李四都会养猪，种土豆。但是李四更擅长养猪，张三更擅长种土豆。这种情况就是他们做自己的不擅长的事情时，成本会很高，他们还是可以从贸易中得到好处。

上面两种情况其实是类似的，术业有专攻，我们可以从贸易中得到好处。但是，**如果张三、李四都擅长养猪，种土豆，这个时候贸易还有必要存在吗？他们还要相互依存吗？**

##  生产可能性

为了探讨上面的问题，我们需要增加点模型的复杂度。我们设张三、李四都=每天工作八小时，并且两人的种土豆、养猪的效率如下：

|      | 生产1kg🐖肉所需时间(min) | 生产1kg土豆所需时间(min) |
| ---- | ----------------------- | ------------------------ |
| 张三 | 20                      | 10                       |
| 李四 | 60                      | 15                       |

张三就是人上人，干什么都比李四效率高。如果两人把精力都用于生产一样东西，就会得到下面这张表（都工作8小时）：

|      | 牛肉 | 土豆 |
| ---- | ---- | ---- |
| 张三 | 24   | 48   |
| 李四 | 8    | 32   |

我们之前学过<u>生产可能性</u>边界这个概念，它说明了一个经济学原理：<u>人们面临着权衡取舍</u>。在这里张三和李四也面临着权衡取舍，生产可能性边界反映了他们面临的各种可能产量的组合：

上图就表示了两个人面临的所有可能，但是这只告诉了他们有哪些选择，却不能告诉他们该怎么选择。为了下面探讨方便，我们假设他们选取了A点和B点。

##  专业化和贸易

有一天，人上人张三找到李四聊会天，大概说了下面这段意思：

> 如果李四你把一天8小时全部用于种土豆，你会得到32kg土豆，你把15kg给我；我把5kg猪肉给你。最后你将得到17kg土豆，5kg猪肉，比原来各多吃了1kg。

可能我们会觉得这样张三一定亏了，但我我们来算一算：

> 张三一天生产18kg猪肉,可以生产12kg土豆，给了李四5kg,还剩13kg；得到李四15kg，就会有27kg土豆。很明显也是赚的：



|                    | 张三                | 李四              |
| ------------------ | ------------------- | ----------------- |
| 无贸易时生产与消费 | 24kg 土豆；12kg猪肉 | 16kg土豆；4kg猪肉 |
| 有贸易时生产       | 12kg土豆；18kg猪肉  | 32kg土豆；0kg猪肉 |
| 有贸易时消费       | 27kg土豆；13kg猪肉  | 17kg土豆；5kg猪肉 |
| 收益               | +3 土豆；+1 猪肉    | +1 土豆；+1猪肉   |

张三、李四两人都可以获益是因为贸易使我们每个人都可以专门从事自己最擅长的工作。

李四将把更多的时间用于种土豆，更少的时间用于养🐖。张三将把更多的时间用于养🐖，更少的时间用于种土豆。由于**专业化和贸易**，我们每个人都可以**不用增加工作时间就消费更多的🐖肉和更多的土豆**。

我们知道张三养🐖和种🥔都擅长，而李李四貌似都不擅长，那么怎么判断李四的“专业”呢？也就是为什么不让李四一天全部养🐖而是让他一整天都种🥔？

这就需要讲下面的概念：绝对优势与比较优势。我们判断一个人的专业，靠的是比较优势而不是绝对优势。

##  比较优势和绝对优势

谁能以较低的成本生产土豆？是张三还是李四？这个问题有两个答案，这是理解贸易所带来的好处的关键。

###  绝对优势

一种方法是比较比较两个生产者所需要的投入。

当比较一个人、一个企业或一个国家与另一个人、另一个企业或另一个国家的生产率时，经济学家用**绝对优势**这个术语。<u>如果生产者生产一种物品所需要的投入较少，就可以说该生产者在生产这种物品上有绝对优势</u>。

在我们的例子中，时间是唯一的投入。我们之前有一张表，上面清清楚楚写了两个人的生产投入：

|      | 生产1kg🐖肉所需时间（min） | 生产1kg🥔所需时间(min) |
| ---- | ------------------------- | --------------------- |
| 张三 | 20                        | 10                    |
| 李四 | 60                        | 15                    |

很明显，张三生产1kg🥔只需要10min，张三生产土豆的绝对优势比较大。

###  比较优势

还有另一种方法：我们不比较生产的投入，而是**机会成本**（得到一样东西所放弃的东西）。我们要知道，多生产一分钟🐖牛肉，就要少生产一分钟的🥔。

- 张三生产1kg的🐖肉，机会成本就是2kg的🥔；生产1kg的土豆，机会成本就是0.5kg的🐖肉。
- 李四生产1kg的🐖肉，机会成本就是4kg的🥔；生产1kg的土豆，机会成本就是0.25kg的🐖肉。

在描述两个生产者的机会成本时，经济学家用**比较优势**这个术语。如果一个生产者在生产X物品时**放弃了较少的其他物品**，即生产X物品的机会成本较小，我们就可以说，他在生产该物品上具有比较优势。

我们很明显就可以看出来，李四生产土豆的机会成本更小；张三生产猪肉的机会成本更小。所以应该像上面那样分配。

> 尽管一个人有可能在两种物品的生产上都具有绝对优势（正如这个例子中的Rose那样），但一个人却不可能在两种物品的生产上都具有比较优势。
>
> 因为一种物品的机会成本是另一种物品机会成本的倒数，如果一个人生产一种物品的机会成本较高，那么，他生产另一种物品的机会成本必然较低。比较优势反映了相对的机会成本。除非两个人有相同的机会成本，否则一个人就会在一种物品上具有比较优势，而另一个人将在另一种物品上具有比较优势。
